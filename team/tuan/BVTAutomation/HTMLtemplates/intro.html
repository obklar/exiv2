EXIV2 BVT AUTOMATION for %BVTName

Exiv2 Build Verification Test Automation is to build/test the Exiv2 automatically.
It will check the latest svn revision of Exiv2 and run build/test if it's necessary.


Current Status

The Last Build: %LastBuild. View log at here.

The Last BVT Run: %LastBVT.

Actions

1. Click me to check latest SVN Revision and run BVT.
Note: it may take a few seconds before the web page is redirected to the the BVT page. Please be patient

2. To view the log file for the revision x, access to %domainname/archive/x
You can also use the keyword "latest" in the places of x to view the log for latest version. e.g. %domainname/archive/latest

3. To run BVT for the revision x, access to %domainname/bvt/x
e.g. %domainname/bvt/%LastBuild

Notes
The BVT is only runned for the revision whose svn info contains [build:all] or [build:...,linux,...]. Hence, please enter one of these keywords in your commit log message if you want the BVT runned.
